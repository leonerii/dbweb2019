Abstract

Este projeto consiste numa API simples para gerenciamento de registros de filmes. Através da API é possível consultar, editar, criar e deletar um registro através de métodos HTTP. Exemplo de uso: curl -X POST http://127.0.0.1:3000/filmes -d '{"title": "qualquer um", "year": 2019}' Irá criar um filme com o título "qualquer um" produzido no ano 2019
